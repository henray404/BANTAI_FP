# Bug & Error Log

Log EVERY bug and error here before fixing it.
One file per bug. Use the template below.

## Why log everything?
- Avoid solving the same bug twice
- Track patterns across sessions
- Help teammates who hit the same issue

## Naming convention
`YYYY-MM-DD_short-description.md`
Example: `2026-05-14_h5py-dll-error.md`

## How to create a new log
Copy `log_template.md`, rename it, fill it in.
